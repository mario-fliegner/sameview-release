# sameview-release

Release assets, Google Play Store listing, screenshots, legal documents, and publication materials for SameView.

## Structure

```
01_Store_Texts     Store listing texts (DE/EN)
02_Graphics        App icon, feature graphic, promo assets
03_Screenshots     Phone and tablet screenshots (DE/EN)
04_PlayConsole     Google Play Console configuration
05_Releases        Release-specific notes and history
06_Legal           Privacy Policy, Terms, Imprint
07_Review_Assets   Assets and notes for Google Play review
```

This repository contains all non-source-code assets required to publish and maintain SameView on Google Play.
